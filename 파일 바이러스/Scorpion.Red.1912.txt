Scorpion.Red.1912

다른 이름 : Red_Scorpion.1912
제작지    : 국산
국내 발견 : 1996년11월 이칠영
종류      : 파일
기억 장소 : 상주형
감염 방법 : 기생형
암호 방식 : 암호화
감염 위치 : COM, EXE

감염 증상 : 

22시 이후 파일을 감염시킨 후 메시지 출력

내용      : 

MDIR III 2.72A 크랙으로 속이고 올라온 프로그램(파일 이름: MC-V272A.EXE, 파일 크기: 3803 바이트)에 포함된 바이러스이다. 사용자들의 호기심을 악용한 대표적인 사례로 MDIR 사용자가 많은 만큼 감염/피해 규모도 광범위했다.

상주 후 파일 열기 및 실행되는 COM, EXE 파일을 감염시키며 감염된 파일의 크기는 1912 바이트 늘어난다. 단, 파일 이름이 'IO', 'OS', 'SC', 'WI', 'V', 'T', 'CO' 'F-'로 시작하는 파일이나 윈도우 파일은 감염시키지 않는다. 

만일 22시 이후에 실행되는 파일이 있다면 감염시킨 후 화면에 붉은 색 테두리를 만들면서 다음과 같은 메시지를 출력하지만 너무 빨리 지나가기 때문에 테두리선이 번쩍이는 것만 보인다.

제작지    : 국산
국내 발견 : 1995년01월 홍수민
종류      : 파일
기억 장소 : 비상주형
감염 방법 : 기생형
감염 위치 : EXE

감염 증상 : 

도스 기능 수행시 장애 발생

내용      : 

감염 파일이 실행되면 현재 디렉토리에서 EXE 파일을 찾아 감염시킨다. 감염 파일의 작성 시간과 파일속성은 변하지 않으며, 감염된 파일의 크기는 753 바이트 늘어난다. 

매주 월요일마다 바이러스 일부분이 기억장소에 상주하며, 이후 도스 프로그램을 사용할 때마다 내부 값을 변경시켜 이 값이 0이 되면 도스 기능에 다음과 같은 장애가 발생한다.

파일생성         -> path not found 에러 메시지 출력
파일삭제         -> 파일이 안지워짐 
속성 얻기/바꾸기 -> 읽기전용으로 설정됨
디렉토리 생성    -> 디렉토리가 안 만들어짐
디렉토리 삭제    -> 디렉토리가 안지워짐
디렉토리 이동    -> 이동이 안됨

감염 파일 내부에는 다음과 같은 문자열이 포함되어 있다.

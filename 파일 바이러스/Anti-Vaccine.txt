Anti-Vaccine

다른 이름 : Corea, KOV
제작지    : 국산
국내 발견 : 1995년03월 윤병훈
종류      : 파일
기억 장소 : 비상주형
감염 방법 : 기생형
감염 위치 : COMMAND.COM, COM, EXE

감염 증상 : 

백신 프로그램 삭제, 메시지 출력

내용      : 

COM, EXE 파일을 감염시키는 파일 바이러스로 감염 파일이 실행되면 기억장소의 최상위에 자신을 재배치하는데 재배치 과정에서 시스템이 정지되는 경우도 발생한다. 

감염된 후 사용중인 디스크의 루트디렉토리에서부터 파일을 찾는다. 만약 현재 디렉토리에서 감염시킬 파일을 찾지 못하면 모든 디렉토리를 뒤져서 찾는다. 이 과정은 3개의 파일이 감염되거나 모든 실행파일이 감염된 것을 확인할 때까지 계속된다. 따라서 프로그램 실행 시간이 
지연된다.

파일을 찾을 때는 모든 파일을 찾으며 만약 파일 이름이 "V3, TV, KA" 일 경우 해당 파일을 삭제한다. 이때 파일 확장자는 검사하지 않기 때문에 "V3, TV, KA"로 시작되는 모든 파일이 삭제된다. 만약 백신 파일들을 삭제시킬 때 시간이 밤 11시이며 '삑' 하는 소리와 함께 "Korea 
VIRUS 1995.3"라는 메시지를 출력한다.

COM 파일은 1000 - 62000 바이트 사이의 파일만을 감염시키며, 감염된 파일의 크기는 926 바이트 늘어난다. EXE 파일은 1000 바이트 파일을 감염시키며, NE 형태(윈도우즈, OS/2)이거나 내부 오버레이 파일인 경우 감염시키지 않는다. 감염된 파일의 크기는 1038 바이트 
늘어난다.

일단 감염 파일은 파일 작성 초가 62초가 되며 62로 감염 여부를 검사하므로 파일을 62초로 만드는(비엔나, 대만, 방랑자.II 등) 바이러스에 감염 파일은 감염되지 않는다.

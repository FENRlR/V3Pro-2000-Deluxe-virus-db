IVP.687

다른 이름 : IVP.Paethon
제작지    : 외산
국내 발견 : 1997년07월 김형섭
종류      : 파일
기억 장소 : 비상주형
감염 방법 : 기생형
암호 방식 : 암호화
감염 위치 : COM, EXE

감염 증상 : 

감염 파일 실행 후 메시지 출력

내용      : 

감염 파일이 실행되면 현재 디렉토리부터 루트디렉토리까지 옮기면서 모든 EXE, COM 파일을 감염시킨다. 감염된 파일의 크기는 687 바이트 늘어나지만 해당 파일의 날짜, 시간, 파일속성은 변하지 않는다. 

단, 파일 이름 중에서 6, 7번째 글자가 "N"이나 "D"인 파일(예 COMMAND.COM)은 감염에서 제외된다. 감염 파일이 실행되면 다음과 같은 메시지가 출력된다.

"THE REAL! EXTINCTOR .oO PHAETHON Oo. [IVP]"

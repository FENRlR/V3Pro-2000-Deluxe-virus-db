Quicky

다른 이름 : Quicky.1376, Quicksilver
제작지    : 외산
국내 발견 : 1995년12월 이경남
종류      : 파일
기억 장소 : 상주형
감염 방법 : 기생형
암호 방식 : 암호화
감염 위치 : EXE

내용      : 

EXE 파일을 감염시키는데 에러 처리를 하지 않아 쓰기방지된 드라이브에서 파일을 실행하면 쓰기방지 에러가 발생한다.

EXE 헤더와 실제 파일길이를 검사해서 일치하지 않으면 감염시키지 않으므로 오버레이 기법을 사용하는 EXE 파일을 삭제하지 않는다. 이 바이러스는 해당 인터럽트를 가로채고 자체적으로 사용하지 않는 기능도 포함하고 있어 외부의 어떤 프로그램에서 호출될 때만 작동한다.

바이러스 내부에 체크섬 내용을 담은 파일들을 삭제하는 부분이 있으나 실제로는 작동하지 못한다. 간단한 암호화 기법을 사용하며, 암호를 풀어보면 다음과 같은 문자열이 포함되어 있다.

"CHKLIST.TAV QUICKY"

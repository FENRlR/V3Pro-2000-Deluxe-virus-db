Scorpion.Blue.1902

다른 이름 : Blue_Scorpion.1902
제작지    : 국산
국내 발견 : 1996년11월 김일순
종류      : 파일
기억 장소 : 상주형
감염 방법 : 기생형
암호 방식 : 암호화
감염 위치 : COMMAND.COM, COM, EXE

감염 증상 : 

22시 이후 파일을 감염시킨 후 화면 테두리선을 그리면서 메시지 출력

내용      : 

MDIR III 2.72A 크랙으로 속이고 올라온 프로그램(파일 이름: MC-V272A.EXE, 파일 크기: 3770 바이트)에 포함된 바이러스로. 사용자들의 호기심을 악용한 사례로 MDIR 사용자가 많은 만큼 감염 및 피해 규모도 광범위했다. 

단순한 암호화 바이러스 형태로 백신의 검색을 혼란을 주기 위해 데이터를 조작했으며, 감염 파일 실행시 해당 디렉토리내에 모든 EXE 파일을 감염시킨다. 상주 후 파일 열기 및 실행되는 COM, EXE 파일을 감염시키며, 감염된 후 1902 바이트 늘어나는데 파일 이름이 'IO', 
'OS', 'SC', 'WI', 'V', 'T', 'CO' 로 시작하는 경우 감염시키지 않는다. 

만일 22시 이후에 실행되는 파일이 있다면 감염시킨 후 화면에 파란색 테두리를 만들면서 다음과 같은 메시지를 출력하지만 너무 빨리 지나가기 때문에 테두리선이 번쩍이는 것만 보인다.

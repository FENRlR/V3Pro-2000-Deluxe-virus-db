Scorpion.Red.2081

다른 이름 : Red_Scorpion.2081
제작지    : 국산
국내 발견 : 1996년12월
종류      : 파일
기억 장소 : 상주형
감염 방법 : 기생형
암호 방식 : 암호화
감염 위치 : COM, EXE

감염 증상 : 

23시 이후 파일을 감염시킨 후 화면 테두리를 붉은 색으로 만들고 메시지 출력

내용      : 

감염 파일 실행시 해당 디렉토리내에 모든 EXE 파일을 감염시키며, 상주 후 파일 열기 및 실행되는 COM, EXE 파일을 감염시키며, 감염된 파일의 크기는 2081 바이트 늘어난다. 단, 파일 이름이 'CO', 'IO', 'OS', 'SC', 'WI', 'V', 'T' 로 시작하는 경우 감염시키지 않는다. 

만일 23시 이후에 실행되는 파일이 있다면 감염시킨 후 화면에 붉은 색 테두리를 만들면서 다음과 같은 메시지를 출력하지만 너무 빨리 지나가기 때문에 테두리선이 번쩍이는 것만 보인다.

제작지    : 국산
국내 발견 : 1998년06월 우연희
종류      : 파일
기억 장소 : 상주형
감염 방법 : 기생형
감염 위치 : COMMAND.COM, COM, EXE

감염 증상 : 

메시지 출력

내용      : 

감염 파일이 실행되면 기억장소에 상주한 후 실행되는 COM, EXE 파일을 감염시킨다. 단, 파일 이름이 "V3, TV, T2, ND, WP, AN, OT, LL"로 끝나거나 파일 크기가 1239 바이트 이하인 파일은 감염되지 않는다.

감염된 파일 크기는 1239 바이트 늘어나지만 파일 속성, 파일 작성 날짜는 변하지 않고 파일 작성초가 22초로 변한다. 또한, MEM/C명령으로 메모리 상태를 보면 COMMAND.COM이 두 번 상주한 것처럼 보인다.

은폐형 바이러스로 바이러스가 기억장소에 상주해 있으면 감염된 후 늘어난 파일 크기를 정상적인 길이로 보여 사용자를 속인다. 도스의 DIR 명령으로는 증가된 파일 크기를 알 수 있지만 MDIR 같은 프로그램에서는 정상적인 파일 크기로 보인다. 

감염된 후 특징적인 증상으로 파일 5개를 감염시키면 다음과 같은 메시지가 출력된 후 파일이 실행된다.

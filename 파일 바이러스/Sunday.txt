Sunday

다른 이름 : Jerusalem.Sunday
제작지    : 외산 미국 1989년11월
국내 발견 : 1990년10월 신창윤
종류      : 파일
기억 장소 : 상주형
감염 방법 : 기생형
감염 위치 : COM, EXE

감염 증상 : 

일요일에 실행되는 파일 삭제

내용      : 

1989년 11월 미국의 워싱턴주에 있는 시애틀(Seattle)에서 처음 발견되었는데 예루살렘 바이러스의 변형으로 EXE 파일에 재 감염되지는 않는다. 제작자의 의도는 1990년 이후에 일요일마다 감염 파일을 삭제하는 것이나 프로그램 자체의 실수로 1990년 이후에도 파일 감염만 
시도한다.

COMMAND.COM 파일을 제외한 모든 COM, EXE 파일을 감염시킨다. 감염된 COM 파일 크기는 1636 바이트 늘어나고 EXE 파일 크기는 16의 배수로 된 후 1636 바이트 늘어난다. 감염 파일 내부에는 다음과 같은 문자열이 포함되어 있다.

"Today is SunDay! 
 Why do you work so hard?..
 All work and no play make you a dull boy!.
 Come on! Let's go out and have some fun"

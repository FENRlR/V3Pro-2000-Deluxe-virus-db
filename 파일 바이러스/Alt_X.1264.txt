Alt_X.1264

제작지    : 국산
국내 발견 : 1998년01월 조춘구
종류      : 파일
기억 장소 : 상주형
감염 방법 : 기생형
감염 위치 : COMMAND.COM, COM, EXE

감염 증상 : 

비정상적인 키보드 입력 현상

내용      : 

해킹 버전인 V31200.EXE 파일로 위장하여 사설 BBS를 통해 확산된 바이러스로 외형상 최신 버전 V3+인 것처럼 사용자를 속였다. 감염된 파일의 크기는 1264 바이트 늘어나며, 감염 파일의 작성 날짜가 감염 당시 날짜로 변경된다.

읽기전용속성 파일인 경우는 감염되지 않으며, 파일 이름이 다음과 같은 파일도 감염되지 않는다.

 1. '.' 전 5번째부터 두 글자가 'MM'인 경우 
    예) COMMAND.COM
 2. '.' 전 2번째부터 두 글자가 'V3' 'TV' 'T2' 'WP'로 시작하는 경우
    예) V3.EXE TV.EXE T2.EXE HWP.EXE
 3. '.' 전 5번째부터 두 글자가 'V3' 시작하는 경우
    예) V3RES.EXE
 4. '.' 전 4번째부터 두 글자가 'SC' 로 시작하는 경우
    예) SCAN.EXE

감염된 후 특징적인 증상은 바이러스가 메모리에 상주한지 30분 후부터 1분 간격으로 다음과 같은 키가 눌린 것 같은 증상이 나타난다.

"alt+A, alt+L, alt+X, alt+\, alt+F4, ENTER"

감염 파일 내부에는 다음과 같은 문자열이 포함되어 있다.

"Hello ~~ my name is ALT-X virus.."

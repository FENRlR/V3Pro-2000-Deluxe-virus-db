제작지    : 국산
국내 발견 : 1997년07월 김정대
종류      : 파일
기억 장소 : 상주형(은폐기법 사용)
감염 방법 : 기생형
감염 위치 : COMMAND.COM, COM, EXE

감염 증상 : 

메시지 출력

내용      : 

은폐기능을 사용하며, COM 파일의 경우 파일 앞부분에, EXE 파일의 경우 파일 뒷부분에 감염된다.

메모리 상주한 후 파일 열기, 파일속성 설정, 프로그램 실행 및 파일 관련 도스 내부 명령어(Copy, Type) 등을 사용할 때 감염되며, 감염된 파일의 크기를 1530 바이트 줄여 보여주는 은폐기능을 가지고 있다.

하드 디스크의 용량을 DIR 명령이나 기타 유틸리티로 보고자 할 경우 해당 디렉토리에서 파일 크기가 9892 바이트 이상인 COM 파일만 감염되며, 그 외의 경우는 해당 파일 크기가 9892 바이트 이상인 파일만 감염된다. 

이 때 파일 작성시간 중에서 초가 58초와 62초인 경우는 감염에서 제외되며, 감염 파일의 작성 시간은 58초 혹은 62초가 된다.

감염 파일의 실행 날짜가 1995년이 아니거나 1995년이라면 5월 이후 또는 1월에서 4월중 15일 이전인 경우 CMOS 정보를 파괴한다. 또한, 파일이 감염된 시간이 오전 8시 이전에 매시간 6분인 경우 다음과 같은 메시지가 출력된다.

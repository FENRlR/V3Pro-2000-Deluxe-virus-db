종류      : 파일
기억 장소 : 상주형

암호 방식 : 암호화
감염 위치 : COMMAND.COM, COM

감염 증상 : 

하드 디스크 데이터 파괴

내용      : 

감염 파일 실행 후 실행되는 COM(COMMAND.COM 파일 포함)파일만 감염시킨다. 감염된 파일의 크기는 1065 바이트 늘어나지만 해당 파일의 날짜, 시간, 속성은 변하지 않는다.

감염 파일 실행 시간이 시스템 시간으로 6/100초이거나 매년 5월 1일 실행되면 하드 디스크의 데이터를 파괴시킨다. 또한, 부트섹터에 트로이목마 프로그램을 복사하므로 매년 5월 하드 디스크로 부팅할 경우 하드 디스크 일부 영역의 데이터를 파괴한다. 

감염된 파일 내부에는 다음과 같은 문자열이 포함되어 있다.

"[YCWRomeo79.51] [YCWR] [Since 1979] Corea"

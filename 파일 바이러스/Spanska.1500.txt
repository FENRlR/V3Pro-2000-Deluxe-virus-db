Spanska.1500

다른 이름 : Spanska
제작지    : 외산
국내 발견 : 1997년05월 박행운
종류      : 파일
기억 장소 : 비상주형
감염 방법 : 기생형
암호 방식 : 암호화
감염 위치 : COM, EXE

감염 증상 : 

그래픽 출력, 매시 30분에 30초 이내에 파일 실행시 시스템 사용 불능

내용      : 

동일인지 제작한 것으로 보이는 외국산 파일 바이러스로 COM, EXE 파일을 감염시키며, 파일 이름의 3, 4번째가 'MM'이면 해당 파일은 감염시키지 않는다. 

감염된 파일의 크기는 1500 바이트 늘어나며, 감염 파일의 날짜와 파일속성은 변하지 않는다. 감염 파일이 실행되면 현재 디텍토리부터 모든 디렉토리를 찾아다니면서 COM, EXE 파일을 각각 3개씩 감염시킨다.

파일 중에서 총 6개를 감염시켰거나 더 이상 감염시킬 파일이 없으면 시간을 검사하여 매시 30분이고 30초 이내이면 화면이 마치 스크린 세이버처럼 목성 표면이 움직이는 듯한 그림이 출력된다. 이 동작이 시작되면 멈출 수 없기 때문에 시스템을 다시 부팅해야만 한다.

또한, 증상이 나타나는 화면 상단에는 다음과 같은 메시지가 출력된다.

"Mars Land, by Spanska
 (coding a virus can be creative)"

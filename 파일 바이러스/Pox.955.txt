Pox.955

다른 이름 : Npox.955, EVILGEN-1
제작지    : 외산
국내 발견 : 1996년04월 이동수
종류      : 파일
기억 장소 : 상주형(은폐기법 사용)
감염 방법 : 기생형
감염 위치 : COMMAND.COM, COM, EXE

감염 증상 : 

매월 24일 웜 부팅시 하드 디스크 포맷, CHKDSK.EXE 실행시 FAT 에러, CHKDSK 명령시 파일 파괴

내용      : 

기억장소에 상주하면서 C 드라이브의 루트디렉토리에 위치하는 COMMAND.COM 파일을 먼저 감염시킨다. 은폐기법(Stealth Technique)을 사용하는 바이러스로 기억장소에 바이러스가 상주하고 있으면 감염 파일의 길이가 늘어나지 않은 것처럼 만든다. 

감염된 COM, EXE 파일의 크기는 955 바이트 늘어나지만 파일 이름이 'PROT'(F-PROT.EXE), 'SCAN'(SCAN.EXE), 'LEAN'(CLEAN. EXE)로 끝나는 파일은 감염되지 않는다. 

감염된 후 특징적인 증상은 매월 24일 웜 부팅을 하면 하드 디스크를 포맷한다. 또, CHKDSK.EXE를 실행하면 FAT 에러가 발생하므로 CHKDSK/F와 같은 명령을 하면 파일이 파괴된다. 감염 파일 내부에는 다음과 같은 문자열이 포함되어 있다.

"C:\COMMAND.COM NuKE Pox V1.1 - R.S"

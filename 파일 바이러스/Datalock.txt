Datalock

제작지    : 외산 미국 1990년11월
국내 발견 : 1991년09월
종류      : 파일
기억 장소 : 상주형
감염 방법 : 기생형
감염 위치 : COMMAND.COM, COM, EXE

감염 증상 : 

DBF 파일을 열면 "out of file handles" 에러 발생

내용      : 

23000 바이트 이상의 COM, EXE 파일을 감염시키며 파일 크기가 920 바이트 늘어나는데 감염된 후 DBF 파일을 열면 "out of file handles"라는 에러 메시지가 출력된다. 감염 파일 내부에는 다음과 같은 문자열이 포함되어 있다.

"DataLock version 1.00"

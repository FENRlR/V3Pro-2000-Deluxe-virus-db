Dead.1364

제작지    : 외산
국내 발견 : 1997년04월 최병원
종류      : 파일
기억 장소 : 상주형
감염 방법 : 기생형
감염 위치 : COMMAND.COM, COM, EXE

감염 증상 : 

특정시간에 디스크의 데이터 파괴

내용      : 

감염 파일이 실행되어 감염되었을 때 현재 시스템에서 감염 파일의 경우 아무런 문제가 없으나 외부에서 최초로 감염시킨 파일을 256번 실행시키면 256번째 실행될 때 하드 디스크의 데이터를 파괴한다.

감염 파일이 실행되면 메모리에 상주하고 COMMAND.COM 파일을 먼저 감염시킨 후 활동을 시작한다. 파일 제작, 파일 열기, 파일 종료, 파일 실행시 감염되며 읽기전용속성 파일도 감염된다.

감염된 파일의 크기는 1364 바이트 늘어나지만 원래 파일 크기가 16의 배수가 아니면 1 - 15 바이트의 쓰레기 데이터가 파일에 추가되어 실제 파일 크기는 1379 바이트까지 늘어날 수도 있다. 감염 파일의 날짜와 시간은 변경되지 않는다.

제작지    : 국산
국내 발견 : 1997년10월 강병수
종류      : 파일
기억 장소 : 상주형
감염 방법 : 기생형
암호 방식 : 암호화
감염 위치 : COM

감염 증상 : 

데이터 파괴, 실행 장애

내용      : 

감염 파일이 실행되면 메모리에 상주하며, 이후 실행되는 COM 파일만 감염된다. 감염된 파일의 크기는 681 바이트 늘어나지만 해당 파일의 날짜, 시간, 속성은 변하지 않는다.

감염 파일이 실행된 시간이 시스템 시간으로 6/100초이면 하드 디스크 데이터를 파괴시키며, 매월 25일 감염 파일이 실행되면 해당 프로그램이 실행되지 않고 종료된다. 감염 파일 내부에는 다음과 같은 문자열이 포함되어 있다.

"[NuKE] N.R.L.G. AZRAEL"

VBS/CoolNotepad

제작지    : 외산 2000년05월
종류      : 파일

감염 증상 : 

메시지 출력

내용      : 

VBS/CoolNotepad는 외국산 스크립트 웜으로 2000년 5월 29일 발견되었다. 아직 국내엔 보고된 바 없다. 이 웜은 MS 아웃룩이나 mIRC를 이용해 자신을 전파할 뿐 다른 파일을 감염시키지는 않는다.

감염된 VBS 파일을 실행하면 스크립트가 실행되고 COOL_NOTEPAD_DEMO.TXT.vbs 파일이 생성되며 레지스트리를 변경해 다음 번 부팅때도 자동으로 실행되도록 한다.

MS 아웃룩(MS Outlook)을 이용해 아웃룩 주소록 등록된 주소로 웜이 포함된 메일을 보낸다.

보내지는 메일의 제목과 내용은 다음과 같다.

제목 : "Cool Notepad Demo" 
내용 : "Hey check out this text file I sent it will do something neat 
        in notepad. Enjoy :-)" 

mIRC(인터넷 채팅 프로그램)이 설치된 폴더에 스크립트 파일을 만든다. 이후 mIRC로 IRC(Internet Relay Chat. 인터넷을 통해 구축된 대화방) 채팅을 할 때 같은 방 있는 다른 사람들에게 감염된 파일을 자동으로 전송한다.

특별한 증상은 없으며 다만 일종의 스팸으로  전자메일 서버 부하를 일으킬 수 있다.

Scream.696

다른 이름 : Scream.II.696, Fist.696, Screaming_Fist.696
제작지    : 외산 캐나다 1994년03월
국내 발견 : 1994년03월 유채준
종류      : 파일
기억 장소 : 상주형
감염 방법 : 기생형
암호 방식 : 암호화
감염 위치 : COMMAND.COM, COM, EXE

내용      : 

공개용 프로그램에 다수 감염되어 미국에서 많은 피해가 발생한 것으로 알려져 있는 변형 바이러스로 감염 파일이 실행되면 기억장소에서 상주한 후 COM, EXE 파일을 감염시킨다.

감염 파일이 실행되면 기억장소에서 상주 여부를 검사하여 기억장소에 상주한 후 파일 열기, 실행, 파일속성 변경 등이 되는 COM, EXE 파일을 감염시킨다. 기억장소에 상주하면서 C 드라이브의 루트디렉토리에 있는 COMMAND.COM을 먼저 감염시킨다.

감염된 후 기본 메모리는 2 킬로바이트 줄어들며, UMB 값이 0으로 되는데 길이가 300 - 64000 바이트인 COM, EXE 파일을 감염시킨다. 감염된 COM 파일 크기는 696 바이트 늘어나고 EXE 파일의 경우 512의 배수가 된 후 696 바이트 늘어나므로 원래 파일 길이에 따라 696 - 
1207 바이트 늘어난다.

감염 파일 내부의 암호를 풀어보면 다음과 같은 문자열이 포함되어 있다.

"C:\COMMAND.COM Screaming Fist II"

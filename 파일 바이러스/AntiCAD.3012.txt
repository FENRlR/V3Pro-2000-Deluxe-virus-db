AntiCAD.3012

다른 이름 : Plastique, Plastique.3012
제작지    : 외산 대만 1989년
국내 발견 : 1997년09월 김홍운
종류      : 파일
기억 장소 : 상주형
감염 방법 : 기생형
감염 위치 : COM, EXE

감염 증상 : 

하드 디스크의 데이터 파괴, 메시지 출력

내용      : 

COM(COMMAND.COM 파일 제외), EXE 파일을 감염시기며, 감염된 파일의 크기는 3012 바이트 늘어난다.

ACAD.EXE 프로그램을 실행시키면 다음과 같은 메시지를 출력한 후 캐드 프로그램의 실행파일인 ACAD.EXE 파일을 실행되지 않도록 하는 증상을 갖고 있다.

감염된 후 특징적인 증상은 바이러스가 메모리에 상주하면 컴퓨터 속도가 느려지며, 감염된 상태에서 웜 부팅(<Alt>+<Ctrl>+<Del>)을 시도하면 화면이 320x200으로 바뀌고 경고음을 출력한 후 하드 디스크의 데이터를 파괴한다.

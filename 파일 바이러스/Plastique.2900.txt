Plastique.2900

다른 이름 : AntiCAD.2900.Plastique.A, Jeru-5 Jerusalem.Taiwan.2900
제작지    : 외산 대만 1990년06월
국내 발견 : 1991년06월 송승영
종류      : 파일
기억 장소 : 상주형
감염 방법 : 기생형
감염 위치 : COM, EXE

감염 증상 : 

CMOS 정보 파괴, 시스템 속도 저하, 4분 30초마다 음악 연주

내용      : 

오토캐드(ACAD.EXE) 파일을 삭제하는 특징 때문에 안티캐드(AntiCAD) 바이러스로 불리기도 하며, 감염 파일이 실행되면 기억장소에 상주한 후 실행되거나 파일열기 등을 하는 COM, EXE 파일을 감염시킨다. 단, COMMAND.COM 파일은 감염에서 제외된다. 

감염된 COM 파일 크기는 2900 바이트 늘어나며, EXE 파일은 16의 배수가 된 후에 2900 바이트 늘어나므로 원래 파일 길이에 따라서 2900 - 2915 바이트 늘어난다. 또한 읽기전용속성 파일도 감염시킬 수 있으며 감염 파일속성이나 작성 시간은 변하지 않는다.

ACAD.EXE 파일을 실행하면 A, B, C, D 디스크의 모든 데이터를 삭제하고, CMOS 정보 역시 파괴한 후 시스템 동작을 정지시킨다. 감염 파일을 실행시킨 지 31일 후에 또다시 실행시키면 시스템 속도가 느려지거나 4분 30초마다 음악을 연주한다.

이때 웜 부팅(<Ctrl>+<Alt>+<Del>)을 시도하면 속도가 느려지는 경우에는 A, B, C, D 디스크의 데이터를 모든 삭제하고, CMOS 정보를 파괴한 후 시스템을 정지시키며, 음악이 연주되던 경우에는 CMOS 정보만 파괴한 후 시스템을 정지시킨다.

또한, 4000번 이상 키보드를 누를 때까지 디스크에 자료를 저장하지 않았을 경우에는 자료가 저장되지 않게 만든다. 내부에 암호화된 부분을 풀어보면 다음과 같은 문자열이 포함되어 있다.

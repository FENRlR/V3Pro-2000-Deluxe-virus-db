Rosebud

제작지    : 국산
국내 발견 : 1994년06월 이해석
종류      : 파일
기억 장소 : 상주형
감염 방법 : 기생형
감염 위치 : EXE

감염 증상 : 

월 + 일 = 30인 경우 실행되는 파일 삭제

내용      : 

EXE 파일만 감염되며, 읽기전용속성 파일과 파일 이름이 V3, SCAN, CLEAN으로 시작하는 파일은 감염시키지 않는다. 또한 A, B 드라이브에서 있는 파일은 감염되지 않는다. 이 바이러스에 감염되면 감염된 파일의 크기가 16의 배수가 된 후 912 바이트 늘어나므로 감염된 
파일의 크기는 912 - 927 바이트 늘어난다.

감염된 후 특징적인 증상은 월과 일을 더해서 30이 되는 날에 실행되는 파일들을 삭제한다. 감염 파일 내부에는 다음과 같은 문자열이 포함되어 있다.

"WARNING : This Rosebud Virus is simple, 
 because it was made for interest.
 But Next virus will be bit more complex."

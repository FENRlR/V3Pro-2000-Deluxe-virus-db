IVP.790

제작지    : 외산
국내 발견 : 1997년06월 이병진
종류      : 파일
기억 장소 : 비상주형
감염 방법 : 기생형
암호 방식 : 암호화
감염 위치 : COMMAND.COM, COM, EXE

감염 증상 : 

매월 20일 메시지 출력 후 하드 디스크 데이터 파괴 및 시스템 정지

내용      : 

감염 파일이 실행되면 현재 디렉토리부터 루트디렉토리까지 옮기면서 모든 COM, EXE 파일을 감염시킨다. 감염된 파일의 크기는 790 바이트 늘어나지만 해당 파일의 날짜, 시간, 파일속성은 변하지 않는다. 

단, 파일 크기가 64745 바이트 이상인 COM 파일에 감염되면 해당 파일이 파괴되어 실행되지 않는다. 감염된 시간이 1993년 이후이고, 매월 20일 이후면서 오전 11시 이후라면 화면에 [IVP] 라는 메시지를 출력하고 하드 디스크의 데이터를 파괴한다.

이후 시스템에서 "틱틱" 하는 소리가 나며 시스템이 정지한다. 감염 파일 내부에는 다음과 같은 문자열이 포함되어 있다.

"A Test Virus!Devil"

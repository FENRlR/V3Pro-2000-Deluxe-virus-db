No_Import_Rice

제작지    : 국산
국내 발견 : 1994년04월 윤은영
종류      : 파일
기억 장소 : 상주형
감염 방법 : 기생형
감염 위치 : EXE

감염 증상 : 

메시지 출력

내용      : 

감염 파일이 실행되면 메모리에 상주하며 4월 이후 월요일인 경우 실행, 속성 변경, 파일열기시 파일 크기가 10001 바이트 이상인 파일만 감염시킨다.

메모리에 상주한 후 디렉토리를 만들거나, 삭제, 변경, 실행 종료를 총 10회 실행한 후 디렉토리를 이동하려고 할 때 "path not found"라는 에러 메시지가 출력되고, 디렉토리를 만들거나 지우려고 할 때 "access denided"라는 에러 메시지가 출력된다. 또한, 감염된 프로그램을 
실행 종료하면 다음과 같은 메시지가 출력된다.

">>> No Import Rice <<<"
 
또한, 감염 파일 내부에는 다음과 같은 문자열이 포함되어 있다.

"(c)KoRea-PeaSant
 COMSPEC=
 >>>NoImportRICE!<<<$"

Spanska.1120

다른 이름 : Spanska.1120.B
제작지    : 외산
국내 발견 : 1997년03월
종류      : 파일
기억 장소 : 비상주형
감염 방법 : 기생형
암호 방식 : 암호화
감염 위치 : COM

감염 증상 : 

매시 52분 20초 미만일 때 그래픽 출력

내용      : 

이 바이러스는 외국산 바이러스로 자신을 Spanska로 칭하는 사람이 만든 파일 바이러스이다. 외국에서는 1997년 3월 발견되었으며 한국에서도 동일한 시기에 발견되었다.

비상주형 파일 바이러스로 COM 파일만 감염시키며, 감염 파일이 실행되면 현재 디렉토리와 상위 디렉토리에서 6개의 COM 파일을 감염시킨다. 파일을 찾는 도중 ANTI-VIR.DAT나 CHKLIST.MS 파일이 존재하면 삭제하는데 감염된 파일의 크기는 1120 바이트 늘어난다. 

암호화 바이러스로 간단히 자신을 암호화하며, 감염된 시스템의 시간이 52분 20초 미만일 때 화면 상단에 "To Carl Sugan, poet and scientist,"라는 메시지를 화면 하단에 "this little Cosmos. (Spanska 97)"라는 메시지를 출력하고 별들(흰점)이 다수 지나간다.

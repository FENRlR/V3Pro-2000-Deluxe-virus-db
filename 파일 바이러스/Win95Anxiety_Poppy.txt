Win95/Anxiety_Poppy

제작지    : 외산 1997년11월
국내 발견 : 1997년11월 정상욱
종류      : 파일
기억 장소 : 상주형
감염 방법 : 기생형
감염 위치 : EXE

감염 증상 : 

부정기적인 시스템 정지

내용      : 

한국에서 최초로 발견된 윈도우 95 전용 바이러스로 최초의 윈도우 95용 바이러스로 알려진 보자(Boza) 바이러스와는 달리 한글 윈도우 95에서도 정상적으로 작동하며 기억장소에 상주하기 때문에 파일 감염 속도가 상당히 빠르다.

감염 파일이 실행되면 기억장소에 상주한 후 윈도우나 도스 창에서 파일열기 되는 PE 포맷의 EXE 파일을 감염시킨다. 한편, 윈도우 NT용 EXE 파일을 감염시킬 수는 있으나, 감염 파일은 윈도우 NT에서 동작하지 않는다.

제작자의 의도는 감염 파일이 실행될 때마다 임의의 레지스트리를 파괴하는 것이나 바이러스의 버그로 인해 실행되지는 않는다. 참고로 메모리 상주시 버그로 인해 시스템이 자주 정지되는 현상이 발생하기도 한다.

Scream.709

다른 이름 : Stranger, Fist.709, Screaming_Fist.709
제작지    : 외산 미국 1992년07월
국내 발견 : 1992년11월
종류      : 파일
기억 장소 : 상주형
감염 방법 : 기생형
암호 방식 : 암호화
감염 위치 : COMMAND.COM, COM, EXE

내용      : 

감염 파일이 실행되면 기억장소에 상주여부를 판단하여 기억장소에 상주하며 실행, 파일 열기, 파일 이름 및 파일속성이 변경되는 COM, EXE 파일을 감염시킨다. 기억장소에 상주하면서 C 드라이브의 루트디렉토리에 있는 COMMAND.COM 파일을 먼저 감염시킨다. 

상주한 후 300 - 62000 바이트의 COM, EXE 파일을 감염시키는데 오버레이 파일은 감염시키지 않는다. 감염된 COM 파일 크기는 709 바이트 늘어나고, EXE 파일은 512의 배수가 된 후에 709 바이트 늘어나기 때문에 원래 파일 길이에 따라 709 - 1220 바이트 늘어난다. 

단, 파일 작성 날짜는 변하지 않는다. 다른 변형 바이러스들과 달리 감염된 후 기본 메모리를 줄이지 않는다. 감염 파일 내부의 암호를 풀어보면 다음과 같은 문자열이 포함되어 있다.

"I am a stranger in a strange land...
 C:\COMMAND.COM"

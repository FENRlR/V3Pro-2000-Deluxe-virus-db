Thursday_12th

다른 이름 : CD
제작지    : 외산 독일
국내 발견 : 1997년05월 곽창훈
종류      : 파일
기억 장소 : 상주형
감염 방법 : 기생형
암호 방식 : 암호화
감염 위치 : COM, EXE

감염 증상 : 

12일의 목요일 화면에 메시지 출력

내용      : 

COM(COMMAND.COM 제외), EXE 파일을 감염시키는 파일 바이러스로 감염된 파일의 크기는 2161 - 2175 바이트 늘어나지만 파일의 날짜, 시간, 파일속성은 변하지 않는다. 

파일 이름 중에서 'SCAN', 'CLEAN', 'VIR', 'ARJ', 'FLU', 'COMMAND'라는 글자가 있으면 감염되지 않으며, 감염 파일이 실행되면 메모리에 상주한 후 파일 열기, 실행 등의 작업을 할 때 해당 파일을 감염시킨다. 

감염된 후 특징적인 증상은 12일의 목요일에 감염 파일이 실행될 때 다음과 같은 메시지가 출력된다. 단, 매번 출력하는 것이 아니라 4번째 파일마다 출력되며 키보드 입력을 기다린 후 원래 프로그램이 실행된다.

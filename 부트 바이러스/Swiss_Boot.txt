Swiss_Boot

다른 이름 : Swiss_Army
제작지    : 외산
국내 발견 : 1997년05월 장길현
종류      : 부트
기억 장소 : 상주형(은폐기법 사용)
감염 위치 : 플로피 디스크의 부트섹터, 하드 디스크의 도스부트섹터

감염 증상 : 

기본 메모리 3 킬로바이트 줄임, 매년 2월 7일 디스크의 데이터를 파괴한 후 메시지 출력

내용      : 

플로피, 하드 디스크의 도스부트섹터를 감염시키는 외국산 부트 바이러스로 감염되면 기본 메모리의 크기는 3 킬로바이트 줄어든다. 부팅할 때부터 플로피 디스크 드라이브에 있던 디스켓은 감염되지 않고 디스켓의 교환작업이 있거나 새로 넣은 디스켓만 감염된다. 

감염된 후 특징적인 증상으로 매년 2월 7일 디스크의 앞부분 내용을 겹쳐 쓰므로 저장되어있던 데이터가 파괴되며, 다음과 같은 메시지와 함께 '삑' 소리를 출력한다.

"Schafft die Schweizer Armee ab !"
